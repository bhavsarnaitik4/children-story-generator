# Children's Story Generator Backend
This is the backend API for the Children's Story Generator, powered by a custom-trained GPT model.

## Tech Stack
- **FastAPI**: Backend Framework
- **PyTorch**: Model Inference
- **SQLAlchemy**: User & Story Database
- **PostgreSQL**: Production Database (Supabase)
- **Docker**: Containerization for Hugging Face Spaces
